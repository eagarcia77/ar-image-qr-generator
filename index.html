Generador de Imágenes para Realidad Aumentada

El profesor pega el URL directo de una imagen alojada en Blackboard Content Collection. El sistema valida el enlace, crea una experiencia AR y genera un QR Code seguro.

1. Datos de la experiencia

Debe ser un enlace directo a una imagen JPG, PNG, WEBP, GIF o SVG. Si Blackboard pide login, el estudiante deberá estar autenticado.

Normalmente se completa automáticamente. Debe terminar en viewer.html.

2. Resultado

Pega el URL de una imagen y presiona “Generar QR Code”.

El QR aparecerá aquí.

Abrir experiencia

Por qué se corrigió

Los enlaces de Blackboard Content Collection muchas veces contienen signos como ?, &, = y tokens largos. Si el URL se pega directamente dentro de otro URL, el QR puede romperse y el visor recibe el mensaje “no hay contenido”. Esta versión codifica toda la información en un parámetro seguro llamado data.

También se añadió validación visual. Si la imagen no carga en la vista previa, el QR tampoco funcionará para el estudiante. En ese caso, debes revisar permisos, autenticación o si el enlace realmente apunta a una imagen.
