Generador de Realidad Aumentada INTER SG

Crea experiencias de Realidad Aumentada con imágenes, videos MP4/WebM o enlaces de YouTube. El sistema genera un QR Code y utiliza el marcador INTER SG.

Corrección importante

Esta versión no depende del botón de “Probar contenido” para crear el QR. Aunque la prueba falle por permisos de Blackboard, puedes generar el QR si el enlace está escrito correctamente.

1. Crear experiencia AR

Para video AR real se recomienda un archivo directo .mp4 o .webm. Para YouTube, la experiencia AR mostrará la miniatura y un botón para abrir el video. Si esta página ya está publicada, déjalo como aparece.

Instrucciones para publicar en Blackboard

  1. Pega el URL del contenido.
  2. Selecciona si es imagen, video directo o YouTube.
  3. Presiona Probar contenido si deseas validar el recurso.
  4. Presiona Generar QR Code.
  5. Descarga el QR Code.
  6. Descarga el marcador INTER SG.
  7. Sube ambas imágenes a Blackboard.
  8. Indica al estudiante que escanee el QR y apunte la cámara al marcador INTER SG.

No publiques solamente el QR. El estudiante también necesita el marcador INTER SG para activar la Realidad Aumentada.
