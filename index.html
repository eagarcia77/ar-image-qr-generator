Generador de Realidad Aumentada INTER SG

Programa integrado para trabajar con Image, Video MP4/WebM, YouTube, PDF document y Web link. Genera una sola imagen descargable con el QR Code y el Marker INTER SG en el centro. Para YouTube, el sistema abre el video directamente en YouTube para evitar bloqueos del reproductor embebido. En celular o tableta, el contenido aparece como una tarjeta flotante y permite zoom con los dedos. Cuando el contenido es una imagen PNG transparente, se muestra con fondo transparente.

Funciones

Esta versión integra las funciones del generador de imágenes/videos: fondo transparente para PNG, Zoom In/Zoom Out, Activar video, Abrir video en YouTube, Mostrar contenido sin marcador y diagnóstico para Blackboard.

Cómo se usa en Blackboard

  1. Genere la imagen única.
  2. Descargue el archivo PNG.
  3. Publíquelo en Blackboard o imprímalo.
  4. El estudiante escanea el QR Code y luego apunta al Marker INTER SG del centro.
  5. Si el marcador no detecta, usa Mostrar contenido sin marcador.

Para archivos de Blackboard, revise que el Content Collection tenga permisos Student/Course Users → Read.
