Generador Universal AR INTER SG

Programa integrado para trabajar con Image, Video MP4/WebM, YouTube, PDF document y Web link. Ahora puede crear dos tipos de imagen: QR + Marker integrados y QR + Marker separados en una sola imagen. También incluye una Opción C con estilo visual más moderno para el QR y una nueva opción Tiger Style – INTER SG inspirada en rayas de tigre. También añade opciones de Marker INTER y Marker HIRO para uso genérico.

Funciones principales

Incluye fondo transparente para PNG, Zoom In/Zoom Out, Activar video, Abrir video en YouTube, Mostrar contenido sin marcador, diagnóstico para Blackboard y dos formatos de imagen única.

1. Crear experiencia AR

INTER es una opción genérica para otras unidades de la Inter. HIRO es una opción totalmente genérica y universal de AR.js. Use enlaces directos del Blackboard Content Collection. Los estudiantes deben tener permiso de Read. Si esta página ya está publicada, déjela como aparece. Modern usa una presentación visual más limpia. Tiger Style – INTER SG crea un QR personalizado con detalles visuales inspirados en rayas de tigre, manteniendo la legibilidad. Inter Premium – Simple QR crea una opción súper personalizada en estilo Inter, con colores institucionales y un QR más simple para escanear.

Cómo se usa en Blackboard

  1. Genere las dos imágenes.
  2. Descargue la versión que prefiera: integrada o separada.
  3. Publique el archivo PNG en Blackboard o imprímalo. Seleccione también el tipo de Marker adecuado antes de generar las imágenes.
  4. El estudiante escanea el QR Code y luego apunta al Marker seleccionado: INTER SG, INTER o HIRO.
  5. Si el marcador no detecta, usa Mostrar contenido sin marcador.

Para enlaces largos como YouTube o Web link, normalmente la versión separada escanea mejor. Para archivos de Blackboard, revise que el Content Collection tenga permisos Student/Course Users → Read.
