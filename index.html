Generador Universal AR

Programa integrado para trabajar con Image, Video MP4/WebM, YouTube, PDF document y Web link. Ahora permite escoger entre Marker INTER SG, Marker INTER y Marker HIRO.

Opciones genéricas de Marker

Use Marker INTER SG para San Germán, Marker INTER para otras unidades de la Inter, o Marker HIRO para una opción totalmente genérica. El programa ahora sí cambia el Marker en la imagen generada y en el visor AR.

Uso recomendado

  1. Seleccione el Tipo de Marker.
  2. Genere las dos imágenes.
  3. Publique la versión integrada o separada en Blackboard.
  4. El estudiante escanea el QR y apunta al mismo Marker seleccionado.

Para otras unidades de la Inter, use Marker INTER. Para una opción totalmente genérica, use Marker HIRO.
