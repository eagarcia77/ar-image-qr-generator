Generador de Realidad Aumentada INTER SG

Herramienta verificada para trabajar con imágenes, videos MP4/WebM y YouTube. Está optimizada para enlaces del Content Collection de Blackboard porque muestra el contenido como capa HTML sobre la cámara, evitando problemas comunes de CORS en WebGL.

Importante sobre Blackboard Content Collection

El enlace debe ser directo al archivo y los estudiantes deben tener permiso de Read. Si Blackboard exige login, el estudiante debe estar autenticado para que el recurso cargue. Esta herramienta no rompe ni evita permisos de Blackboard.

Checklist antes de publicar en Blackboard

  1. El archivo del Content Collection tiene permisos de Read para Student o Course Users.
  2. El enlace abre directamente el archivo, no una página intermedia.
  3. El QR abre la experiencia en HTTPS.
  4. El marcador INTER SG se publica completo, sin recortar bordes.
  5. El estudiante escanea el QR, permite la cámara y apunta al marcador INTER SG.

Si el marcador no se detecta en algunos celulares, la experiencia incluye un botón de respaldo para mostrar el contenido sin marcador.
