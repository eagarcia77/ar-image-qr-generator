Generador de Realidad Aumentada Inter SG

Permite crear códigos QR para experiencias de Realidad Aumentada usando imágenes o videos. Luego el profesor descarga el QR Code y el marcador Inter SG para publicarlos en Blackboard.

1. Crear experiencia AR

Debe ser un enlace directo a una imagen o video. Para videos se recomienda MP4. Si esta página ya está publicada, déjalo como aparece.

2. Descargar recursos

Después de generar el QR, descarga estas dos imágenes y publícalas en Blackboard:

  1. QR Code
  2. Marcador Inter SG

Instrucciones rápidas para el profesor

  1. Suba la imagen o el video al Content Collection de Blackboard o a un hosting accesible.
  2. Copie el URL directo del archivo.
  3. Pegue el URL en este generador.
  4. Presione Probar contenido.
  5. Presione Generar QR Code.
  6. Descargue el QR Code.
  7. Descargue el marcador Inter SG.
  8. Publique ambas imágenes en Blackboard junto con las instrucciones para el estudiante.

Importante: el estudiante necesita el QR Code para abrir la experiencia y el marcador Inter SG para activar la Realidad Aumentada.
