Actividad de Realidad Aumentada INTER SG

Para completar esta actividad, necesitas usar dos recursos: el QR Code y el marcador INTER SG.

  1. Escanea el QR Code con tu celular o tableta.
  2. Permite el acceso a la cámara cuando el navegador lo solicite.
  3. Abre o imprime el marcador INTER SG.
  4. Apunta la cámara hacia el marcador INTER SG.
  5. Observa la imagen, video o recurso de YouTube asociado.

Importante: el QR Code abre la experiencia y el marcador INTER SG activa el contenido aumentado.
