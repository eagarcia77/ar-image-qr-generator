# Generador AR PDF/Link INTER SG

Programa separado para crear experiencias de Realidad Aumentada con documentos PDF y enlaces.

## Compatible con
- PDF documents
- Web links
- Blackboard Content Collection links

## Archivo principal
Abra `index.html`.

## Nota
Los documentos PDF de Blackboard pueden requerir login. Si el PDF no se muestra embebido, use el botón Abrir documento.
