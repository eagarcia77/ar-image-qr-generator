Generador AR PDF/Link INTER SG

Programa para crear experiencias AR con PDF documents y web links. En esta versión, el Marker INTER SG se coloca en el centro del QR Code con tamaño controlado para mantener la lectura del QR y se descarga una sola imagen.

Importante

La imagen descargada reúne el QR y el Marker en un solo archivo. Esto funciona mejor si la imagen se imprime o se muestra en otra pantalla para que el estudiante la escanee con su celular o tableta.

Cómo usar la imagen única

  1. Genere la imagen única.
  2. Descargue el archivo PNG.
  3. Publíquelo en Blackboard o imprímalo.
  4. El estudiante escanea el QR Code. Luego apunta la cámara al Marker INTER SG que está en el centro de la misma imagen.

Recomendación: para mejor experiencia, muestre la imagen en una computadora o imprímala, y use un celular aparte para escanear y detectar el Marker. El Marker fue reducido para que el QR pueda seguir escaneando correctamente.
