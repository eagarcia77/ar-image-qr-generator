Generador AR PDF/Link INTER SG

Programa para crear experiencias de Realidad Aumentada con PDF documents y web links. Está optimizado para enlaces del Blackboard Content Collection.

Uso recomendado

El QR abre una experiencia con cámara, el marcador INTER SG activa una tarjeta flotante y el usuario puede ver o abrir el PDF/enlace. Si Blackboard bloquea la vista embebida, el botón Abrir documento funciona como respaldo.

Checklist para Blackboard

  1. El documento PDF está en Content Collection.
  2. El documento tiene permiso Student / Course Users → Read.
  3. El enlace abre el documento directamente.
  4. El QR abre la experiencia en HTTPS.
  5. El marcador INTER SG se publica completo.

Si Blackboard bloquea la vista embebida del PDF, el usuario debe usar el botón Abrir documento.
